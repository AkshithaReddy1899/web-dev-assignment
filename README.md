# web-dev-assignment

## Description

A simple react photo gallery

## Built With

- React
- Redux toolkit
- tailwind css

## Getting started

1. Clone the project `git clone https://github.com/AkshithaReddy1899/web-dev-assignment`
2. Installing the dependencies by running `npm install`
3. To start the server `npm start`

### Prerequisites

- JavaScript
- React
